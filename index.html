on passe au niveau suivant

flex

this is the shortland for flex-grow, flex-shrink and flex-basis combined.The second and third parameters ( flex-shrink and flex-basis) are optional.Default is 0 1 auto.

flex grow

The defines the ability for a flex item to grow if necessary.It accepts a unitless value that serves as a proportion.It dictates what amount of the available space inside the flex container the item should take up.

flex-shrink

This defines the ability for a flex item to shrink if necessary. Negative numbers are invalide.

flex-basis

This defines the default size of an element before the remaining spaces is distributed. It can be a lenght (e.g.20%,5rem,etc...) or a keyword.The auto keyword mean "look at my width or height property".
